# Politique de sécurité

## Signaler une vulnérabilité

Merci de ne pas publier publiquement une vulnérabilité potentielle dans les issues GitHub.

Utilisez le formulaire de contact officiel en précisant qu’il s’agit d’un signalement de sécurité :

https://www.cereya.fr/contact

Fournissez, dans la mesure du possible :

- l’URL concernée ;
- une description claire du problème ;
- les étapes permettant de le reproduire ;
- son impact potentiel ;
- toute capture utile ne contenant pas de données personnelles.

## Divulgation responsable

Nous demandons de laisser un délai raisonnable pour analyser et corriger le problème avant toute communication publique.

## Périmètre de ce dépôt

Ce dépôt est documentaire et ne contient pas le code source des plateformes Cereya.

Les signalements portant uniquement sur le contenu éditorial peuvent également être transmis via le formulaire de contact.
